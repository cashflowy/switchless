# Welcome to Switchless
<Subtitle>
Switchless is Cashflowy's React UI component library.
</Subtitle>
Switchless is an open-source library of React components that uses MUI/Joy for foundational styling and design.

## Getting Started

To use these components in your project, you can import them from the <strong>switchless</strong> package

```jsx
import { ComponentName } from 'switchless';
```

Replace `ComponentName` with the name of the component you want to use.


## Quick Example

Here's a simple example of using a Switchless component:

```jsx
import { Link } from 'switchless';

function MyComponent() {
  return <Link href="/">Home</Link>;
}
```
