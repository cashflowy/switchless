# Welcome to Switchless Component Library
<Subtitle>
Switchless is an open-source library of React UI components adhering to the Cashflowy design language.
</Subtitle>
Switchless is an open-source library of React UI components built on top of MUI/Joy and designed specifically for Next.js 14's app router architecture,
Developed with the Cashflowy design language in mind. 
Switchless serves as the backbone for Cashflowy and its products, ensuring consistency, performance, and flexibility in UI development.

## Getting Started

To use these components in your project, you can import them from the 'switchless' package:

```jsx
import { ComponentName } from 'switchless';
```

Replace `ComponentName` with the name of the component you want to use.


## Quick Example

Here's a simple example of using a Switchless component:

```jsx
import { Link } from 'switchless';

function MyComponent() {
  return <Link href="/">Home</Link>;
}
```
## Key Features

- **Next.js 14 Optimized**: Designed for seamless integration with the app router
- **MUI/Joy Foundation**: Built on top of trusted UI libraries
