## Usage

```jsx
import { NumberFormatter } from 'switchless';

<NumberFormatter value={1234567} style="US" />
```
## Props

The NumberFormatter component accepts the following props:

- `value`: (required) The number to be formatted
- `style`: (optional) The formatting style, either 'US' or 'IN'. Defaults to 'US'
- `precision`: (optional) The number of decimal places to display. Defaults to 2

The user can decide the precision of the decimal .
Note: The precision is only applicable for the decimal part of the number.


#### Lower Precision
