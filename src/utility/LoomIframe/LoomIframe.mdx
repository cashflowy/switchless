

## Props
- `src`: (required) The URL of the Loom video share link

## Implementation Details

The `LoomIframe` component is built using MUI Joy components and includes the following features:

1. Responsive layout with a 16:9 aspect ratio for the embedded video.
2. Icon button to open the video in a new tab.
3. Automatic conversion of share URL to embed URL for cleaner video display.
