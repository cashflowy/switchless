# MoreOptionsMenu Component 

<Subtitle>
  A iframe component for displaying Google Sheets
</Subtitle>



```jsx

import { MoreOptionsMenu } from 'switchless';

<MoreOptionsMenu
 
 onEdit={handleEdit}
 onDelete={handleDelete}
 itemName="comment" 
 />

```

## Props
- `onEdit`: (required) OnEdit function to be passed
- `onDelete`: (required) OnDelete function to be passed
- `itemName`: (optional) The itemName of the delete Modal
