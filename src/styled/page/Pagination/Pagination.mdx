# Pagination Component 

<Alert startDecorator={<InfoIcon />} sx={{ mb: 2 }} color="info" variant="soft">
  The Pagination component is designed to work seamlessly with Next.js 14 app router. It automatically handles URL updates for page navigation.
  The component will update the URL with the selected page number.
</Alert>


```jsx
import { Pagination } from 'switchless';

<Pagination pageCount={10} />
```

## Props

The Pagination component accepts the following props:

- `pageCount`: (required) The total number of pages.


## Usage Notes

```jsx
const MyPage = () => {
  return (
    <div>
      <h1>My Paginated Content</h1>
      {/* Your content here */}
```
