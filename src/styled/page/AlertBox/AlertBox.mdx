# Alert Box


## Example Usage

```jsx
import {AlertBox} from 'switchless';
export default function MainComponent() {
    return <AlertBox
        title="Data missing"
        message="No Data to show. Adjust filter or create data."
        icon={false}  // Default is true
    />
}
```
