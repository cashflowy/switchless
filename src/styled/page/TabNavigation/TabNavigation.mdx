## Overview

The `TabNavigation` component provides a customizable tab interface for navigating between different sections of your application. It supports both horizontal and vertical layouts and can display optional count badges for each tab.

## Basic Usage

```jsx
import { TabNavigation } from 'switchless';
const tabs = [
{ header: 'Tab 1', count: 5 },
{ header: 'Tab 2', count: 0 },
{ header: 'Tab 3', count: 10 },
{ header: 'Tab 4' },
];
<TabNavigation tabHeaders={tabs} />
```

## Usage Notes

1. The component automatically updates the URL based on the selected tab, making it easy to share links to specific tabs.
2. Tab headers are converted to URL-friendly formats (lowercase, spaces replaced with hyphens).
3. The component is responsive and supports horizontal scrolling on smaller screens.
4. The componnent updates the URL. To avoid navigation issues Create desired routes or redirect to the desired routes.

## Props

The TabNavigation component accepts the following props:

- `tabHeaders`: (required) An array of tab objects with `header` and optional `count` properties.
- `vertical`: (optional) Boolean to set the orientation of tabs. Defaults to false (horizontal).
