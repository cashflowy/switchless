# CustomStepper Component 

<Subtitle>
  A Stepper component for displaying Google Sheets
</Subtitle>

```jsx

import { CustomStepper } from 'switchless';

const steps = [
  { title: 'Order Placed', status: 'completed' },
  { title: 'Processing', status: 'pending' },
  { title: 'Shipped', status: 'pending' },
  { title: 'Delivered', status: 'pending' }
]

<CustomStepper steps={steps}/>

```

## Props
- `steps`: (required) An Array Of Object, with the title and the completion status.
