## Overview

The `PageHeader` component is designed to provide a consistent and customizable header for your pages. It includes support for breadcrumbs, a main title, and optional action buttons.

## Basic Usage




```jsx
import { PageHeader } from 'switchless';

<PageHeader header="Dashboard"/>
```

## Props

The `PageHeader` component accepts the following props:

- `header`: (required) The main title of the page. Can be:
  - A string for a simple header
  - An object with `part1` and `part2` properties for a two-part header with different styling
- `headerLevel`: (optional) The heading level for the main title. Defaults to 'h3'.
- `breadcrumbs`: (optional) An array of breadcrumb objects with `text` and optional `href` properties.
- `RightButtons`: (optional) A component or function that returns action buttons for the header.

## Header 

### Header as a string

You can pass a string to the `header` prop.

```jsx
<PageHeader header="Simple Header" />
```
PageHeader is rendered as a simple header with a title.

<Canvas of={PageHeaderStories.simpleHeader} />


### Header as an object

You can pass an object to the `header` prop to create a two-part or three-part header.
The object can have `part1`, `part2`, and `part3` properties. Each with a different styling hardcoded in the component.

<b>Note:</b> it only supports up to three parts.

```jsx
const header = { part1: 'Three ', part2: 'Part', part3: 'Header' }

<PageHeader header={header} />

```
